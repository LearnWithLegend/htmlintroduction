HTML Detail

HTML ya Hyper Text Markup Language web pages banane ke liye istemal hota hai. HTML web page ki structure ko describe karta hai aur aik series of elements se consist karta hai1. HTML elements browser ko ye batate hain ke content ko kis tarah display karna hai1. HTML mein kuch basic elements hain:


  1. !DOCTYPE html: Ye line browser ko batati hai ke ye document HTML5 hai.
  2. html: Ye tag HTML document ki shuruat aur khatam hone ki jagah batata hai.
  3. head: Ye section page title, meta information, CSS, JavaScript, etc. ke liye hota hai.
  4. title: Ye tag document ka title specify karta hai jo browser tab mein show hota hai.
  5. body: Ye tag actual content ko contain karta hai jaise ke headings, paragraphs, images,


hyperlinks, tables, lists, etc3. HTML mein tags aur attributes ka istemal hota hai. Tags content ko surround karte hain aur uski nature define karte hain, jabke attributes additional information provide karte hain about the HTML elements3. HTML seekhna asaan hai aur aap apni website khud bana sakte hain3. HTML coding ki practice karne ke liye aap online editors ka istemal kar sakte hain3. HTML ki complete references aapko W3Schools par mil jayengi .
