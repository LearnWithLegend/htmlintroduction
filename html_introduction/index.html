HTML

Hyper Text Markup Language

Yeh ek markup language hai jo web pages aur unke content ko structure karne ke liye use hoti hai. HTML elements, jaise ki headings, paragraphs, lists, etc., web pages ke building blocks hote hain.

CSS

Cascading Style Sheets

Yeh ek stylesheet language hai jo HTML ya XML documents ki presentation (jaise ki fonts, colors, spacing, etc.) ko describe karti hai. CSS se hum ek se zyada web pages ka layout ek saath control kar sakte hain.
